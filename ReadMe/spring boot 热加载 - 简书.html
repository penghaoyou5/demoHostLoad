spring boot 热加载

融一 关注
2017.10.30 15:02* 字数 205 阅读 420评论 0喜欢 0

spring boot为了方便开发过程中，不用频繁的重启服务 ，新增了开发者插件，只用简单在pom.xml中添加依赖即可。

    <dependency>
      <groupId>org.springframework.boot</groupId>
      <artifactId>spring-boot-devtools</artifactId>
      <optional>true</optional>
    </dependency>

但是在我们添加完毕之后，我们在修改文件之后，并没有出现热加载的现象，这个问题在eclipse上是没有问题，但是IDEA不行，原始是IDEA并没有在保存代码之后自动编译文件，按照下面两个方式配置完毕即可。

  • 系统设置中修改compiler，勾选上automatically。
image.png
  • 注册
    comand + shift + option + / 打开Maintenance窗口
image.png

选择第一项Registry，勾选红框中的内容即可。

image.png

最终重启服务，修改对应的代码后就可以看到控制台输出restar信息

image.png
